Picoday プライバシーポリシー

Picoday は、1日1枚の写真で毎日を記録するためのアプリです。このページでは、現在の実装における情報の扱いを説明します。

Last updated: 2026-03-20 Terms Support

1. 保存する情報

  • ユーザーが選択または撮影した写真、サムネイル、メモ、記録日付は端末内のアプリ領域に保存されます。
  • 通知設定、テーマ、言語などのアプリ設定は端末内に保存されます。
  • 購入状態は Apple の StoreKit を通じて確認されます。

2. 外部送信と追跡

  • Picoday は、現在の実装でユーザーデータを開発者または第三者のサーバーへ送信しません。
  • 他社アプリや Web サイトを横断するトラッキングを行いません。
  • 広告 SDK、解析 SDK、外部クラッシュレポート SDK は組み込んでいません。

3. 写真と権限

  • 写真ライブラリへのアクセスは、ユーザーが写真を選ぶときだけ使用します。
  • カメラへのアクセスは、アプリ内で写真を撮影するときだけ使用します。
  • 通知権限は任意で、毎日のリマインダーを有効にする場合のみ使用します。

4. 課金について

  • サブスクリプションおよび買い切り課金は Apple の StoreKit を通じて処理されます。
  • Picoday は支払いカード番号などの決済情報にアクセスしません。

5. データ削除

  • 端末内に保存された写真や設定は、アプリ削除または端末側のデータ削除により削除できます。
  • アプリ内で削除した写真は、Picoday の保存領域から削除されます。

6. お問い合わせ

プライバシーに関するお問い合わせは、サポートページからご連絡ください。
